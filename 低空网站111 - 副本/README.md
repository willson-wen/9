# 深圳低空经济研究小组网站

## 项目说明
本网站致力于提供EVTOL（电动垂直起降航空器）相关信息，包括公司信息、产品参数、适航认证等资讯。

## 系统架构
- 后端框架：Flask
- 数据库：SQLite
- 前端技术：HTML5, CSS3, JavaScript
- 开发环境：PyCharm
